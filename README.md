# Mobile Shop Stock & Sales Management System

A Java-based web application designed to manage mobile shop inventory, sales, and stock levels efficiently.

## Features
- Add, update, and delete mobile products
- Manage stock quantities
- Record sales transactions
- Automatic stock updates after sales
- Simple admin interface

## Tech Stack
- Backend: Java (Servlets / JSP)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Server: Apache Tomcat
- IDE: NetBeans
